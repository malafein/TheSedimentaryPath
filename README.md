# The Sedimentary Path

The Sedimentary Path is a Valheim BepInEx mod.

## Features

- **The Paths of the Rock and Vine**: Listen closely to the world. It may have something to share with you.
- **Sacred Relics**: Strange stones and potent brews await those who learn the old ways.
- **Hidden Mechanics**: The world reacts to your devotion. Experiment and observe.

## Installation

Install via [r2modman](https://thunderstore.io/package/ebkr/r2modman/) or manually place the DLL in your BepInEx plugins folder.
